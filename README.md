# csci5380-lab9
Automate VM, VN, Docker, and BGP path

This project automates the main tasks required for Lab 9.

## Completed modules
- Virtual network setup for VN-A and VN-B
- Security group setup for ICMP, SSH, TCP, and UDP
- Instance deployment for same-tenant and multi-tenant VMs
- FRR Docker container setup
- Ryu Docker container setup
- Main workflow execution with `main.py`

## Main files
- `network_setup.py`
- `security_setup.py`
- `instance_setup.py`
- `frr_setup.py`
- `ryu_setup.py`
- `main.py`

## Summary
The project demonstrates OpenStack automation for networks, security groups, and virtual machines, together with Docker-based FRR and Ryu BGP setup.
